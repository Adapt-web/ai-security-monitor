# 基于AI的个人网站安全监控系统
## 项目介绍
本项目是一个面向个人网站的轻量级安全监控系统，基于阿里云平台搭建，使用AI工具辅助开发，实现了漏洞扫描、日志分析、自动告警等功能。

## 技术栈
- 操作系统：CentOS 7
- Web服务器：Nginx
- 云平台：阿里云ECS、云防火墙、WAF、云监控
- 编程语言：Python 3.9
- AI工具：ChatGPT 3.5、豆包
